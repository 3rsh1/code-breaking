# Code-Breaking Puzzles dockerfiles && writeups

code-breaking puzzles: https://code-breaking.com/

这里是code-breaking puzzles游戏的所有环境与相关writeup。

运行要求：

- Docker CE 18.0+
- Docker-Compose

运行方法：

```bash
# 进入某个题目目录
cd /path/to/puzzle

# 运行环境
docker-compose up -d

# 环境运行后，在docker-compose.yml中查看端口，访问即可
```
